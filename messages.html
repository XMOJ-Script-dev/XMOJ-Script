登录

无需安装用户脚本，在任意浏览器（包括 iOS Safari）收发 XMOJ 站内短消息。

先在 xmoj.tech 完成登录， 再按下方说明找到您的用户名和会话 ID（PHPSESSID）并填入。

  1. 打开 xmoj.tech 并登录。
  2. 按 F12 打开开发者工具。
  3. 切换到"应用"（Application）标签页。
  4. 左侧展开"Cookie" → 点击 https://www.xmoj.tech。
  5. 在列表中找到 PHPSESSID，复制其"值"列。
  6. 用户名可在页面右上角个人信息处查看。

  1. 打开 xmoj.tech 并登录。
  2. 按 F12 打开开发者工具。
  3. 切换到"存储"（Storage）标签页。
  4. 左侧展开"Cookie" → 点击 https://www.xmoj.tech。
  5. 在列表中找到 PHPSESSID，复制其"值"列。
  6. 用户名可在页面右上角个人信息处查看。

  1. 在菜单栏选择"Safari" → "偏好设置" → "高级"，勾选"在菜单栏中显示开发菜单"。
  2. 打开 xmoj.tech 并登录。
  3. 菜单栏选择"开发" → "显示 Web 检查器"。
  4. 切换到"存储"标签页 → "Cookie" → 找到 PHPSESSID。
  5. 用户名可在页面右上角个人信息处查看。

iOS Safari 不提供直接查看 Cookie 的界面。有以下两种方法：

方法一：使用桌面端获取后手动输入

  1. 在桌面浏览器（Chrome/Firefox/Safari）中按上方说明找到 PHPSESSID。
  2. 将值记录下来，在此处手动填入。

方法二：连接 Mac 使用 Safari Web 检查器

  1. 在 iOS 设置 → Safari → 高级 中，打开"Web 检查器"。
  2. 用数据线将设备连接到 Mac。
  3. 在 Mac 的 Safari → 开发菜单中找到您的设备和 xmoj.tech。
  4. 在存储标签页中找到 PHPSESSID。
书签登录仅适用于桌面浏览器（Chrome/Firefox/Edge），移动端请使用"会话登录"。
第一步：添加书签

将下方按钮拖动到书签栏，或点击"复制代码"后手动新建书签并粘贴为网址。

🔖 登录到短消息 WebUI
第二步：在 XMOJ 上点击书签
  1. 前往 xmoj.tech 并登录。
  2. 点击刚才添加的书签。
  3. 页面会自动跳转回此处并完成登录。
